# Digital Clock

A small, vanilla JavaScript digital clock project that displays the current time in the browser.

## Features

- Displays current time (hours, minutes, seconds)
- Displays current weekday,day,month,year
- 12-hour or 24-hour formatting
- Simple, responsive UI

## Files

- index.html — main HTML file with clock markup
- style.css — styles for the clock UI
- script.js — JavaScript to update and render time
